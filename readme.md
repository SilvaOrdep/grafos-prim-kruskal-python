# Rodar Script:

### Criar venv (Powershell):
python -m venv .venv

### Criar venv (Linux e macOS):
python3 -m venv .venv

### Ativar Venv (Powershell):
.venv\Scripts\Activate.ps1

### Ativar Venv (Linux ou macOS):
source .venv/bin/activate

### Instalar dependências e libs:
pip install -r requirements.txt

### Rodar aplicação Streamlit:
streamlit run main.py
